# 🔥 GaliciaVixía

**Sistema de alerta temperá de incendios forestais en Galicia**

## 🎯 Que é?
Un dashboard en Grafana que monitoriza en tempo real o risco de incendios combinando:
- 🌡️ Temperatura e humidade
- 💨 Velocidade do vento  
- 💧 Séquía acumulada
- 📊 Índice FWI (Fire Weather Index)

## 🚨 Como funciona?
Cando se superan umbrais críticos, o sistema envía alertas automáticas por Telegram/Discord a alcaldes, bombeiros voluntarios e veciños.

## 🛠️ Tecnoloxías
- Grafana Cloud + Grafana Alerting
- MeteoGalicia API (datos tempo real)
- Abertos.Xunta.gal (datos históricos)
- Infinity Plugin
- Bot de Telegram

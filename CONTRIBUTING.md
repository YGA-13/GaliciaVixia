# 🤝 Como Contribuír

Grazas polo teu interese en GaliciaVixía!

## 🚀 Comezar rápido

1. Fork do repositorio
2. `git clone https://github.com/TEU_USUARIO/GaliciaVixia.git`
3. `git checkout -b feature/teua-funcionalidade`
4. Fai os cambios e proba localmente
5. `git push origin feature/teua-funcionalidade`
6. Abre un Pull Request

## 📝 Convencións

- Usa commits descritivos: `feat:`, `fix:`, `docs:`
- Non subas `.env` con tokens
- Proba os cambios antes de enviar

## ❓ Dúbidas?

Abre un Issue en GitHub ou contacta con @YGA-13
